# Twitter AI Bot 🤖🇸🇦
**نظام النشر التلقائي المجدول على X (Twitter) — محتوى ذكاء اصطناعي سعودي**

---

## 📋 نظرة عامة

بوت نشر تلقائي على X يُنشر محتوى عربي متخصص في الذكاء الاصطناعي موجّه للجمهور السعودي، مع ردود تفاعلية على المجتمع التقني.

### الميزات:
- ✅ **3-5 منشورات يوميًا** (أخبار + ثريدات + محتوى تعليمي)
- ✅ **15 رد تفاعلي يوميًا** على هاشتاقات الذكاء الاصطناعي
- ✅ **تنويع تلقائي** في الأسلوب والطول
- ✅ **فلترة صارمة** للمحتوى غير الملائم
- ✅ **مصادر موثوقة** فقط (MIT, Wired, VentureBeat...)
- ✅ **عربية فصيحة** مع ترجمة ذكية للمحتوى الإنجليزي

## ⚙️ الإعداد على GitHub

### 1. إنشاء المستودع
```bash
git init
git add .
git commit -m "🚀 Initial commit — Twitter AI Bot"
git remote add origin https://github.com/YOUR_USERNAME/twitter-ai-bot.git
git push -u origin main
```

### 2. إضافة GitHub Secrets
اذهب إلى: **Settings → Secrets and variables → Actions → New repository secret**

أضف هذه الـ Secrets بالضبط:

| اسم الـ Secret | القيمة |
|----------------|--------|
| `TWITTER_CONSUMER_KEY` | مفتاح المستهلك |
| `TWITTER_CONSUMER_SECRET` | سر المستهلك |
| `TWITTER_ACCESS_TOKEN` | رمز الوصول |
| `TWITTER_ACCESS_TOKEN_SECRET` | سر رمز الوصول |
| `TWITTER_BEARER_TOKEN` | رمز Bearer |
| `TWITTER_CLIENT_SECRET` | سر العميل |
| `OPENAI_API_KEY` | (اختياري) مفتاح OpenAI |

### 3. تفعيل GitHub Actions
- اذهب إلى تبويب **Actions**
- اضغط **"I understand my workflows, go ahead and enable them"**

## 📰 أنواع المحتوى

### 1. أخبار مُترجمة (News)
- مصدر: MIT Tech Review, VentureBeat, Wired, The Verge, TechCrunch
- مُترجمة آليًا للعربية
- مع سؤال تفاعلي وهاشتاقات

### 2. محتوى تعليمي (Educational)
- 20+ تغريدة معدّة مسبقًا
- تدور دوريًا دون تكرار

### 3. ثريدات توعوية (Threads)
- 5 ثريدات كاملة (7-8 تغريدات كل ثريد)
- مواضيع: مبادئ AI، ChatGPT، الوظائف، رؤية 2030، أفضل الأدوات

### 4. حقائق مدهشة (Facts)
- إحصائيات وحقائق AI موجهة للسياق السعودي

### 5. تغريدات قصيرة تفاعلية (Short)
- أسئلة واستطلاعات سريعة لزيادة التفاعل

## 🔧 التخصيص

### إضافة تغريدات جديدة
افتح `content/tweets_pool.json` وأضف:
```json
{
  "id": "edu_XXX",
  "text": "نص التغريدة هنا...",
  "hashtags": "#هاشتاق1 #هاشتاق2"
}
```

### إضافة ثريد جديد
افتح `content/threads.json` وأضف:
```json
{
  "id": "thread_XXX",
  "title": "عنوان الثريد",
  "tweets": ["التغريدة 1", "التغريدة 2", ...]
}
```

### تغيير أوقات النشر
عدّل ملفات `.github/workflows/` وغيّر قيم `cron`

## ⚠️ تحذيرات مهمة

1. **لا ترفع ملف `.env`** على GitHub أبدًا
2. **لا تضع API Keys** مباشرةً في الكود
3. **استخدم GitHub Secrets** دائمًا للبيانات الحساسة
4. راقب حدود API لـ Twitter (Free tier: 500 تغريدة/شهر)
5. تأكد من تفعيل **Read and Write** في إعدادات Twitter Developer
